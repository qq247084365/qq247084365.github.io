1.什么是存储过程？
存储过程就是作为可执行对象存放在数据库中的一个或者多个命令。通俗来讲：存储过程其实就是能完成一定操作的一组SQL语句
2.优点
存储过程允许标准组件式编程
能够实现比较快的执行速度
存储过程减轻网络流量
可被作为一种安全机制来充分利用

begin

as

begin

select * from xuesheng where 学号=@xuehao

end
